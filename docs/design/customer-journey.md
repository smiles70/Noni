# Customer Journey Map

**Version:** 1.0.0
**Owner:** Product

---

## Stages

### 1. Discovery
**Goal:** Learner hears about mynaani and visits the landing page.
**Metrics:** Unique visitors, bounce rate.
**Touchpoints:**
- Word of mouth
- Social media
- Search engines

### 2. Orientation
**Goal:** Learner understands what mynaani is and starts the free curriculum.
**Metrics:** "Start Learning" click-through rate.
**Touchpoints:**
- Landing page hero and introduction
- "How it works" dialog
- Help center

### 3. First Value
**Goal:** Learner completes Module 1 and feels capable using Claude.
**Metrics:** Module 1 completion rate, time to complete.
**Touchpoints:**
- Curriculum pages (context, principle, example, retrieval)
- Progress saving
- Encouragement messages

### 4. Sustained Use
**Goal:** Learner completes Modules 2-3 and develops judgment.
**Metrics:** Modules 2-3 completion rate, return visits.
**Touchpoints:**
- Curriculum menu
- Module transitions
- Help center (common questions)

### 5. Purchase Decision
**Goal:** Learner decides to purchase Modules 4-5.
**Metrics:** Paywall view-to-purchase rate.
**Touchpoints:**
- Paywall page (calm, non-urgent)
- Gift token option
- Pricing transparency ($39 / $59)

### 6. Purchase
**Goal:** Learner successfully buys and accesses Modules 4-5.
**Metrics:** Checkout completion rate, refund rate (<5%).
**Touchpoints:**
- Stripe Checkout
- Purchase confirmation
- Immediate access

### 7. Advocacy
**Goal:** Learner shares mynaani with others.
**Metrics:** Gift purchases, referrals.
**Touchpoints:**
- Gift token purchase ($59 caregiver edition)
- Word of mouth

| Purchase Decision | Paywall -> Purchase | 10% |
| Purchase | Checkout completion | 80% |
| Advocacy | Gift purchases / month | 20 |

## Pain Points

| Stage | Pain Point | Mitigation |
|:---|:---|:---|
| Orientation | "What is Claude?" | Landing page explains in plain language |
| First Value | "I'm too old to learn this" | Geragogy contract: calm, dignified, no urgency |
| Purchase Decision | "Is this a subscription trap?" | One-time purchase, no auto-renew, 30-day refund |
